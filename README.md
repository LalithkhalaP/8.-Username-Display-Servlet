
# 8.-Username-Display-Servlet
Objective:
To display a personalized greeting using a servlet without database operations.
Description:
User enters a username into an HTML form.
The servlet reads it in the service() method and prints:
Welcome, <username>!
This project demonstrates basic HTML-to-Servlet communication using only request and response objects.
Requirements:
HTML:
Form with username input
Servlet:
Override service()
Read request parameter
Print greetings
